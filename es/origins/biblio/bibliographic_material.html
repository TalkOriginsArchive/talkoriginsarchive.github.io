Tema de la bibliografía de TalkOrigins

Materiales bibliográficos

Índice de la bibliografía · /origins/biblio/bibliographic_material.html

Materiales bibliográficos incluye referencias anotadas y fuentes de archivo que documentan la historia de las afirmaciones científicas y pseudocientíficas relacionadas con la evolución, el creacionismo y la arqueología. Esta bibliografía presenta obras como la "Anti-Evolución: Una bibliografía anotada" de McIver, que examina críticamente la literatura anti-evolución. También incluye estudios sobre bases de datos bibliográficas y software, relevantes para comprender la evolución del discurso académico sobre los orígenes y el tiempo profundo.

6. Belic, K. y Surla, D., 2008, Aplicación web amigable para el procesamiento de material bibliográfico: The Electronic Library: v. 26, no. 3: p. 400-410.

DOI: 10.1108/02640470810879536

Resumen

Propósito El propósito de este artículo es la implementación de un sistema de software para el procesamiento de material bibliográfico que no requiere conocimientos sobre ningún formato para la entrada de datos bibliográficos. Esto significa que la entrada puede ser realizada no solo por bibliotecarios, sino también por otras personas, como autores de unidades bibliográficas, estudiantes, empleados, etc. Diseño/metodología/enfoque Se utiliza una metodología orientada a objetos para el desarrollo de sistemas de información mediante herramientas de ingeniería de software asistida por computadora (CASE) y componentes de software. La arquitectura del software es multicapa y basada en la web. La implementación se realiza en un entorno Java. Hallazgos El resultado es una aplicación web mediante la cual los usuarios pueden catalogar material bibliográfico sin estar familiarizados con el formato correspondiente. No obstante, el registro bibliográfico se forma de acuerdo con el formato dado para el procesamiento de material bibliográfico (MARC21, UNIMARC, etc.). Limitaciones/implicaciones de la investigación La generación automática de formularios de pantalla para el conjunto de datos elegido para el procesamiento de material bibliográfico no se proporciona en la versión presentada de la aplicación. Con el fin de eliminar esta limitación, existen soluciones preestablecidas que pueden integrarse en la aplicación. Implicaciones prácticas La aplicación está principalmente destinada a instituciones de investigación que buscan formar su catálogo electrónico y/o bibliografías de investigadores o instituciones. Originalidad/valor La originalidad del artículo radica en la arquitectura del software de la aplicación relacionada con la capa intermedia, es decir, la de lógica de negocio. Esta capa implementa un mecanismo mediante el cual diferentes conjuntos de datos de entrada se mapean a datos persistentes mediante el modelo de objeto único de un formato aceptado para el procesamiento de material bibliográfico (MARC21, UNIMARC u otros).
