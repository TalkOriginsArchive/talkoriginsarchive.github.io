Copyright y uso justo
Un excelente recurso sobre el tema del copyright en general y el concepto de “uso justo”.
  • Todos los documentos en este sitio tienen derechos de autor de sus autores individuales. La colección de estos documentos está protegida por Copyright por The Talk.Origins Archive.
  • Este sitio no tiene objeción a que alguien enlace a cualquier documento en este sitio. Cualquiera puede hacerlo sin pedir permiso.
  • El permiso para reproducir documentos de este Archivo se concede para uso personal o educativo no comercial, siempre que el texto de los documentos no se altere o edite y se incluya una declaración que mencione el copyright del autor o autores. Si una copia se vende por encima del coste o si el texto de un documento se edita de cualquier manera, entonces se debe obtener primero permiso de los autores de ese documento.
    • Si un documento tiene una declaración de copyright contraria a lo anterior, entonces la declaración de copyright de ese documento prevalece sobre este documento en cualquier declaración que contradiga este documento.
    • Si un documento indica que es una reimpresión de algo publicado previamente en otro lugar, entonces debe respetarse el copyright de la fuente original y sus políticas prevalecen sobre este documento.
  • Cualquier copia de uso justo de este Archivo debe ser apropiadamente atribuida. Copiar y pegar sin atribución es simplemente poco ético. El permiso se deniega para cualquier copia de este sitio sin atribución ni al autor o autores ni a este sitio.
  • Cualquier sitio web que cite parte de un documento de este sitio debe enlazar de vuelta al original o proporcionar el URL del original. Simplemente no hay excusa para no hacerlo, ya que enlazar es fácil y lleva muy poco tiempo, y es la netiqueta correcta. El permiso para citar cualquier parte de este Archivo se deniega a cualquier sitio web que se niegue a crear un enlace a o proporcionar el URL de lo que está citando.
  • El Archivo no tiene objeción a la publicación de traducciones de documentos del Archivo y la permite bajo las mismas condiciones para la reproducción de los artículos enumerados arriba siempre que se haga un esfuerzo de buena fe para una traducción precisa, se proporcione un enlace al documento original en inglés, se intente ponerse en contacto con los autores y se respeten sus deseos. Si una traducción se publica en línea, por favor háganoslo saber para que este sitio tenga la opción de enlazar con la traducción.
