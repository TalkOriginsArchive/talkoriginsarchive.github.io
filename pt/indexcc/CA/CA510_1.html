CA510.1: Problemas com a evolução como evidência para a criação.

Alegação CA510.1:

Problemas com a evolução são evidência para o criacionismo.

Resposta:

  1. Esta alegação assume que a criação e a evolução são os dois únicos modelos possíveis, o que é falso. Muitos outros modelos, científicos e de outra forma, foram propostos.

    Problemas mesmo com todos, exceto um, dos modelos não implicam que o modelo restante seja verdadeiro. Outra alternativa é que outra explicação ainda desconhecida esteja correta.

  2. Se a alegação for verdadeira, sua contraparte deveria ser verdadeira pelo mesmo motivo. Os criacionistas estão preparados para aceitar problemas com o criacionismo (que são numerosos) como evidência para a evolução?

  3. A maioria das chamadas problemas com a evolução não são problemas de todo. A maior parte deste site é dedicada a mostrar por quê.
