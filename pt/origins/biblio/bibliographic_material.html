Índice de bibliografia · /origins/biblio/bibliographic_material.html

Materiais bibliográficos inclui referências anotadas e fontes arquivísticas que documentam a história de alegações científicas e pseudocientíficas relacionadas à evolução, criacionismo e arqueologia. Esta bibliografia apresenta obras como "Anti-Evolução: Uma Bibliografia Anotada" de McIver, que examina criticamente a literatura anti-evolução. Também inclui estudos sobre bancos de dados bibliográficos e software, relevantes para compreender a evolução do discurso acadêmico sobre origens e tempo profundo.

6. Belic, K. e Surla, D., 2008, Aplicação web amigável ao usuário para processamento de material bibliográfico: The Electronic Library: v. 26, no. 3: p. 400-410.

DOI: 10.1108/02640470810879536

Resumo

Propósito O propósito deste artigo é a implementação de um sistema de software para processamento de material bibliográfico que não requer conhecimento sobre qualquer formato para entrada de dados bibliográficos. Isso significa que a entrada pode ser feita não apenas por bibliotecários, mas também por outras pessoas, como autores de unidades bibliográficas, estudantes, funcionários, etc. Design/metodologia/abordagem É utilizada uma metodologia orientada a objetos para desenvolvimento de sistemas de informação por meio de ferramentas de engenharia de software assistida por computador (CASE) e componentes de software. A arquitetura de software é multicamada e baseada na web. A implementação é feita em ambiente Java. Resultados O resultado é uma aplicação web por meio da qual os usuários podem catalogar material bibliográfico sem estar familiarizados com o formato correspondente. No entanto, o registro bibliográfico é formado de acordo com o formato dado para processamento de material bibliográfico (MARC21, UNIMARC, etc.). Limitações/implicações da pesquisa A geração automática de formulários de tela para o conjunto escolhido de dados para processamento de material bibliográfico não é fornecida na versão apresentada da aplicação. A fim de eliminar essa limitação, existem soluções pré-definidas que podem ser integradas à aplicação. Implicações práticas A aplicação destina-se principalmente a instituições de pesquisa que visam formar seu catálogo eletrônico e/ou bibliografias de pesquisadores ou instituições. Originalidade/valor A originalidade do artigo reside na arquitetura de software da aplicação relacionada à camada intermediária, ou seja, a de lógica de negócios. Esta camada implementa um mecanismo pelo qual diferentes conjuntos de dados de entrada são mapeados para dados persistentes por meio do modelo de objeto único de um formato aceito para processamento de material bibliográfico (MARC21, UNIMARC ou outros).
