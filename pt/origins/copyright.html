Direitos autorais e uso justo
Um excelente recurso sobre direitos autorais em geral e o conceito de "uso justo".
  • Todos os documentos neste site têm copyright de seus autores individuais. A coleção desses documentos tem copyright do The Talk.Origins Archive.
  • Este site não tem objeção a que alguém faça um link para qualquer documento deste site. Qualquer pessoa pode fazê-lo sem pedir permissão.
  • É concedida permissão para reproduzir documentos neste Arquivo para uso pessoal ou educacional não comercial, desde que o texto dos documentos não seja alterado ou editado e uma declaração mencionando o copyright do(s) autor(es) seja incluída. Se uma cópia for vendida por mais de seu custo ou se o texto de um documento for editado de alguma forma, será necessário obter antes permissão do(s) autor(es) desse documento.
    • Se um documento tiver uma declaração de direitos autorais que contrarie o acima, a declaração desse documento prevalecerá sobre este documento em qualquer parte que contradiga este documento.
    • Se um documento indicar que é uma reimpressão de algo publicado anteriormente em outro lugar, os direitos autorais da fonte original devem ser respeitados e suas políticas prevalecerão sobre este documento.
  • Qualquer cópia sob o princípio de uso justo deste Arquivo deve ter atribuição apropriada. Copiar e colar sem atribuição é simplesmente antiético. A permissão é negada para qualquer cópia deste site sem atribuição ao(s) autor(es) ou a este site.
  • Qualquer site que cita parte de um documento deste site deve voltar ao original por meio de link ou fornecer o URL do original. Não há justificativa para não fazer isso, já que fazer links é fácil e leva pouquíssimo tempo, e é a etiqueta adequada na rede. A permissão para citar qualquer parte deste Arquivo é negada a qualquer site que não queira fazer link para ou fornecer o URL do que está citando.
  • O Arquivo não tem objeção à postagem de traduções de documentos do Arquivo e a permite nas mesmas condições para a reprodução de artigos listadas acima, desde que seja feita uma boa tentativa de tradução precisa, que seja fornecido um link para o documento original em língua inglesa, que haja uma tentativa de contatar o(s) autor(es) e que os desejos do(s) autor(es) sejam respeitados. Se uma tradução for publicada online, por favor avise-nos para que este site tenha a opção de fazer link para a tradução.
