Dicas para pesquisar

[O texto a seguir foi adaptado das instruções de linguagem de consulta do Microsoft Index Server]

Na forma mais simples, uma consulta pode ser apenas uma palavra ou uma frase. Mas com as dicas nesta página, você pode expandir o foco da sua consulta para obter resultados mais completos. Para consultas e exemplos mais complexos, consulte a página de Linguagem de Consulta.

  • Procure por mais de uma palavra. Use os operadores booleanos AND e OR para conectar várias palavras em sua consulta. Por exemplo, digite dog and cat para encontrar apenas documentos que contenham as palavras dog e cat. Digite dog or cat para encontrar todos os documentos que contenham qualquer uma dessas palavras. Não digite dog cat a menos que esteja procurando instâncias onde essas duas palavras apareçam diretamente lado a lado.

  • Procure por palavras com o mesmo prefixo. Por exemplo, no formulário de consulta digite key* para encontrar key, keying, keyhole, keyboard e assim por diante.

  • Pesquise todas as formas de uma palavra. Por exemplo, no formulário digite sink** para encontrar sink, sinking, sank e sunk.

  • Pesquise com a palavra-chave NEAR, em vez de AND, para palavras próximas umas das outras. Por exemplo, ambas estas consultas, system and manager e system near manager, procuram as palavras system e manager na mesma página. Mas com NEAR, as páginas retornadas são classificadas por ordem de proximidade: quanto mais próximas as palavras estiverem, maior será a classificação dessa página.

  • Use parênteses para clarificar a precedência em uma consulta que usa operadores booleanos. Por exemplo, a consulta (dog or cat) and fight encontra documentos contendo a palavra fight e uma de dog ou cat.

  • Coloque aspas em torno das palavras-chave se quiser que o Index Server as trate literalmente. Por exemplo, se você digitar a seguinte consulta:

    "system near manager"

    O Index Server procurará literalmente a frase completa system near manager. Mas se você digitar a mesma consulta sem aspas:

    system near manager

    O Index Server pesquisa todos os documentos para as palavras system e manager na mesma página. Essa busca também incluiria todas as ocorrências da frase system and manager.

  • Refine suas consultas com os termos AND NOT para excluir certos textos da sua busca. Por exemplo, se você quiser encontrar todas as instâncias de surfing, mas não de surfing the Net, escreva a seguinte consulta:

    surfing AND NOT the Net

Essas dicas ajudam a começar, mas para consultas mais complexas e mais exemplos, consulte a página de Linguagem de Consulta.
